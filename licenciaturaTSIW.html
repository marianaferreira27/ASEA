Licenciatura em Tecnologias e Sistemas de Informação para a Web

A Licenciatura em Tecnologias e Sistemas de Informação para a Web tem uma abordagem contemporânea e multidisciplinar, focada no design, conceção e desenvolvimento de produtos e software para a Web. O curso prepara os estudantes para áreas emergentes como computação em cloud, mobilidade digital, e-commerce, UX/UI design e ergonomia digital.

Distingue-se pela integração entre desenvolvimento tecnológico e design de interfaces, promovendo uma forte componente prática e projetual desde o 1.º ano. Além disso, mantém uma ligação próxima ao setor empresarial, contando com diversos parceiros tecnológicos.

A licenciatura destaca-se ainda por ter uma elevada empregabilidade, com estudos indicando uma taxa de desemprego de 0% em 2019, refletindo a forte demanda do mercado por profissionais desta área.

Como sei se esta é a licenciatura certa para mim?

Desenvolvedor Full-Stack: Criação de aplicações web completas (front-end e back-end); Designer de Interfaces (UI Designer): Criação de interfaces visuais intuitivas e eficazes; Especialista em Cloud Computing: Desenvolvimento de soluções baseadas na nuvem; Engenheiro de Software para a Web: Desenvolvimento de soluções digitais inovadoras.
